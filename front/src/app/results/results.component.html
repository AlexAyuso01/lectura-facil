Descripción de la evaluación de los modelos

Cada modelo en esta aplicación analiza pares de frases para determinar su similitud semántica. Las frases se convierten en representaciones numéricas como vectores. Luego, se calcula la similitud entre estos vectores utilizando la similitud del coseno, que varía de 0 a 1 tras convertir los valores negativos en cero.

Cada modelo predice si las frases son semánticamente similares usando un umbral de 0.8. Si la similitud del coseno es mayor que 0.8, las frases se consideran similares (valor de 1). Si es 0.8 o menos, se consideran no similares (valor de 0).

La precisión de un modelo se obtiene comparando las predicciones con las etiquetas reales del conjunto de datos. Si la etiqueta real es 'SI' (las frases son similares), se toma como 1; si es 'NO' (las frases no son similares), se toma como 0.

La precisión del modelo es el porcentaje de predicciones correctas sobre el total, y se muestra junto a cada modelo en la sección 'Modelos usados'."
